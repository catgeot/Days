## 🆕 추가된 메타데이터

모든 100개 여행지에 다음 필드 추가:

| 필드 | 설명 | 예시 |
|------|------|------|
| `tier` | 여행지 등급 (1/2/3) | `1` |
| `popularity` | 인기도 점수 (1-100) | `95` |
| `continent` | 대륙 분류 | `"europe"` |
| `categories` | 다중 카테고리 (배열) | `["urban", "culture"]` |
| `primaryCategory` | 주 카테고리 | `"urban"` |
| `showOnGlobe` | 지구본 표시 여부 | `true` |
| `denseRegion` | 밀집 지역 ID | `"western-europe"` |
