# 통합 작업 계획 (이번 세션 + 이전 과제 결합)

중복 작업을 줄이고 효율성을 극대화하기 위해, 이번에 요청하신 이슈들과 `.ai-context.md`에 명시된 '다음 세션 목표'를 4개의 페이즈(Phase)로 통합하여 재구성했습니다.

## Phase 1: 글로벌 코어, 스마트 AI 검색 고도화 및 최신 모델 마이그레이션
1. **Gemini 3.1 Flash-Lite 마이그레이션**: 가격과 성능이 뛰어난 신규 모델 `gemini-3.1-flash-lite-preview`를 기존 `2.5-flash`가 담당하던 가벼운 작업(스마트 검색어 교정, 툴킷 디펜서, 에러 Fallback 등)에 전면 도입하여 속도와 비용 효율을 극대화합니다.
2. **모바일 줌인(Zoom-in) 방지**: `index.html` viewport에 `maximum-scale=1.0, user-scalable=no` 추가.
3. **마커 연결 오류 수정**: `useHomeHandlers.js`에서 국가명(Country)보다 지명(Name)을 최우선으로 매칭하도록 우선순위 재조정 (그리스->산토리니 오류 해결).
4. **스마트 검색 고도화**: 단순 교정이 아닌, 검색어(예: 핏케언 제도)를 분석해 정확한 `위경도(lat/lng), 지명, 국가`를 JSON 형태로 추출하도록 AI 프롬프트 전면 수정.
5. **유튜브 검색 Fallback 로직**: `fetch-place-videos` Edge Function 수정. 1차 검색 시 `videoCaption: CC` 조건을 제거하고 `여행 브이로그` 키워드와 한국어 우선순위를 적용. 결과가 0개면 조건 없는 `travel vlog`로 일반 검색 수행.

## Phase 2: 스마트 툴킷 통합 최적화 (ToolkitTab.jsx 단일화)
- **외부 링크 모바일 빈 창 문제 (이번 이슈)**: 모바일 환경을 감지하여 링크 클릭 시 새 창(`_blank`)이 아닌 현재 창(`_self`)으로 열리도록 수정하여 뒤로가기로 쉽게 복귀하도록 개선.
- **아웃링크 검색어 고도화 (한/영 혼용 검색 비효율 개선)**: 툴킷 본문에 "센소지(Senso-ji)" 처럼 한글과 영문이 병기될 경우, 스마트 링커(`CopyableWord`) 클릭 시 정규식을 통해 **괄호 안의 영문명만 추출**하고, 여기에 현재 장소의 **영문명(`name_en`)을 결합**(예: `Senso-ji Tokyo`)하여 구글 맵/웹 검색으로 넘기도록 로직을 고도화하여 해외 장소 검색 정확도를 극대화.
- **툴킷 최상단 제목 영역 헤더 겹침(Padding) 해결 (이전 과제)**.
- **디펜서 갱신 시 "툴킷이 없습니다" 노출 버그 수정 (이전 과제)**.
- **로딩 메시지 이원화 (이전 과제)**: 신규 생성 시와 기존 정보 갱신 시 로딩 문구 분리.

## Phase 3: 갤러리 통합 개편 및 네비게이션 고도화 (PlaceGalleryView, PlaceCard/index.jsx)
- **사용자 사진 업로드 기능 신설 (이번 이슈)**:
  - 브라우저 단 이미지 압축(`browser-image-compression`) 적용, Supabase Storage 연동.
  - 업로드 폼 신설 (사진 선택, 캡션 입력, 작가명(프로필 자동/수동), 저작권 동의 체크).
- **다중 API 워터마크 표시 고도화 (이전 과제)**: Pexels, Unsplash, 사용자 업로드 사진별로 출처 명확화.
- **사진 큐레이션(Curation) 개편 (이전 과제)**: 갤러리 큐레이션 버튼 정비 및 사용자 업로드 사진 혼합 배치.
- **메인 네비게이션 3탭 압축 및 토글 (이전 과제)**: 네비게이션을 '리뷰, 위키, 영상' 3개로 압축. 클릭 시 화면이 '갤러리' 모드와 토글되는 방식 도입. 리뷰 탭 하이브리드 헤더 폰트 확대.

## Phase 4: 블로그 인증 유도 UI
- **비로그인 사용자 사이드바 하단 로그인/갤러리 유도 (이번 이슈)**:
  - `Sidebar.jsx`에 비로그인 전용 UI 컴포넌트 추가.
  - 기본 제공되는 여행 썸네일 이미지 슬라이드 배치.
  - "나만의 여행 기록을 남겨보세요", "로그인하고 프로필/갤러리를 관리하세요" 안내 문구 및 [로그인하기] 버튼(`Auth Modal` 호출) 신설.
