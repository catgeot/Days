  - 재생성: `node scripts/extract-travel-spots-list.cjs`

- **Mapbox** 등: 토큰은 **URL 제한·서버/프록시** 관점의 보안 가이드 준수

배포 시: `VITE_SUPABASE_URL`의 **subdomain = project ref** 확인 후, 예)

## 4. AI 협업 핵심 원칙

1. **기획 .md 남발 금지**: 사용자가 **명시적으로** 요청하지 않는 한 `plans/*.md`에 **새 “완료 보고서/대형 계획”**을 만들지 않는다. 필요한 건 **짧은 일지** 또는 **이 파일 5~6절** 갱신.
2. **구조 합의 → 직접 편집**: 가능하면 채팅에 **수백 줄 코드 덤프** 대신, 워크스페이스 **파일을 수정**한다.
3. **UI 임의 변경 금지**: 기존 레이아웃/반응형/모바일 동작을 **승인 없이** 뜯지 않는다. 한국어 문단은 `break-keep` 등 **기존** 규칙 유지.
4. **사용자 확인**: 수정 후 “완료” **단정** 직전, 사용자 **테스트·승인**이 없으면 “제안/대기”로 둔다. 단위 작업 후 **git commit**은 **제안**한다.
5. **고유명사**: `한글명('English name')` 형식 유지.
6. **툴킷·갤러리·제휴**(스마트 링크, 비자/페리, 링크 정합 등) **도메인 로직**은 3절, 해당 **소스·Edge Function**을 **우선**한다.

- **2026-05-07 Home Mapbox 클릭/복귀 UX 정리(완료)**: 지도 클릭 이벤트를 **마커 > 라벨 > 빈 지도(역지오코딩)** 순으로 분리하고, 라벨 클릭은 카드 한글 표기 + URL 영문 slug를 함께 만족하도록 조정. 직접 지도 탐색 클릭은 `focus:false`로 카메라 이동을 막고, 검색/카드/탐색 유입은 `focus:true`로 홈 복귀 시 재포커스를 허용. `pauseRender` 중 누락되던 포커스는 `pendingFocus`로 지연 적용하고, `/place`·`/explore`에서는 지구본 렌더를 확실히 정지. `onIdle` 레이어 재스캔 제거로 뒤로가기/연관키워드 전환 체감 지연 완화. 장소카드 헤더에 홈(지구본) 바로가기 버튼 추가.
- **2026-05-08 Home 글로브 모바일 안정화(진행)**: 모바일에서 마커 탭 이후 빈 지구본 탭이 막히는 회귀를 `HomeGlobeMapbox`/`HomeGlobe`의 hover click-guard 정리로 완화. 현재 위치 버튼은 Mapbox에서 `onGlobeClick` 경로를 재사용해 핀 생성 일관성을 맞춤. 로컬 모바일 Mapbox는 토큰 URL 제한(403, 이후 429+CORS 노출) 영향이 확인되어 DEV 모바일 기본 엔진을 legacy로 두고 QA 진행, 배포 경로는 Mapbox 유지.
